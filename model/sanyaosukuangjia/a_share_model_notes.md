# A股版模型说明

## 模型定位

这是一套面向 A 股的日度状态模型。

核心思想：

- `竞争格局` 看 A 股内部结构是否强
- `流动性` 看外部环境是否支持 A 股
- 最后合成总分，给出市场状态和仓位建议

## 当前总分公式

```text
总分 = 0.55 * 竞争格局分数 + 0.45 * 流动性分数
```

## 竞争格局分数

当前版本只重视 A 股结构，主要由两部分构成：

### 1. 市场主导权

使用字段：

- `raw_ret_5d_csi300`
- `raw_ret_20d_csi300`
- `raw_ret_5d_cyb`
- `raw_ret_20d_cyb`
- `raw_above_ma20_csi300`
- `raw_above_ma60_csi300`
- `raw_above_ma20_cyb`
- `raw_above_ma60_cyb`

公式：

```text
市场主导权 = 0.4 * 短期强弱分 + 0.4 * 中期强弱分 + 0.2 * 趋势强度分
```

### 2. 风格主导权

使用字段：

- `raw_ratio_cyb_sz50`
- `raw_ratio_kc50_hongli`

先计算这两个比值的 20 日变化，再求平均。

如果结果偏高，说明成长风格占优；
如果结果偏低，说明红利和防御风格占优。

### 当前竞争格局总分

```text
竞争格局分数 = 0.25 * 市场主导权 + 0.20 * 风格主导权
```

说明：

- 原始设计里还有主线清晰度、广度/集中度
- 但当前版本这两块未完整接通，所以暂不作为主判断
- 代码会自动对有效项重新分配权重

## 流动性分数

当前版本已经忽略成交额相关字段，因此流动性主要看：

### 1. 全球流动性

使用字段：

- `raw_us2y_yield`
- `raw_us10y_yield`
- `raw_dxy`（当前实际是 `UUP` 代理）

逻辑：

- 美债回落，偏利多风险资产
- 美元走弱，偏利多风险资产

### 2. 中国流动性

使用字段：

- `raw_pbo_c_easing_flag`
- `raw_dr007`（当前通常缺失，可忽略）
- `raw_shibor_3m`（当前解析仍需谨慎）
- `raw_credit_impulse`（当前通常缺失，可忽略）
- `raw_usdcnh`（当前是美元兑人民币牌价代理）

说明：

- `raw_cn10y_yield` 已按当前需求忽略，不参与总分
- 成交额项也已忽略，不参与总分

### 当前流动性总分

```text
流动性分数 = 全球流动性分数 + 中国流动性分数 + 跨资产验证分数
```

但当前主判断主要还是前两项。

## 标签解释

- `lead_market_label`：当前固定按 A 股视角输出，通常为 `A股`
- `lead_style_label`：根据成长 vs 红利风格判断
- `regime_label`：根据总分区间定义市场状态

## 状态区间

```text
总分 >= 75      -> 风险扩张
60 <= 总分 < 75 -> 偏强震荡
45 <= 总分 < 60 -> 中性
30 <= 总分 < 45 -> 偏弱
总分 < 30       -> 风险收缩
```

## 使用建议

这套模型更适合回答三个问题：

1. A 股内部结构强不强
2. 当前外部环境是否支持 A 股
3. 应该偏进攻还是偏防守

它不是单票预测模型，而是市场状态模型。
